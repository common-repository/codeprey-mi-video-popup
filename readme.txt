### Mi Video Popup

Display YouTube or Vimeo or Native video in PopUp, easy to use.

### The Features
1. Easy to install
2. YouTube , Vimeo and native video support.
3. Easy to use.
5. One time video popup when page load at first.
6. Add desired background images or colors.
7. Auto play.
8. Various sizes available.
9. Multiple popups possible.




== Installation ==

### Installation

1. Upload 'mivideopopup' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go in Video popup menu and click on add popup.
4. After creation popup go on add page form 
5. Select popup list from add popup options
